# Fusion Gourmet Landing Page

## Description
This project is a responsive landing page for a gourmet food festival called "Fusion Gourmet". It showcases culinary experiences, event schedules, and multimedia content.

## Technologies Used
- HTML5
- CSS3

## Project Structure
- index.html → Main structure of the website
- styles.css → Styling and responsive design
- img/ → Image assets

## Features
- Semantic HTML5 structure
- Responsive design with media queries
- Flexbox and CSS Grid layout
- Image gallery
- Event schedule section

## How to Run
1. Download or clone the repository
2. Open the folder
3. Double click on `index.html` or open it with a browser
